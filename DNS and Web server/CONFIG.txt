DNS and Web Server Integration - Configuration Commands

# Router Configuration
enable
configure terminal
!
interface g0/0
 ip address 192.168.1.1 255.255.255.0
 no shutdown
!
end
write memory


# Server Configuration (via Packet Tracer GUI)
# 1. Assign static IP:
#    IP Address: 192.168.1.10
#    Subnet Mask: 255.255.255.0
#    Default Gateway: 192.168.1.1
#
# 2. Enable DNS service:
#    - Turn DNS ON
#    - Add record:
#         Name: www.company.local
#         Address: 192.168.1.10
#
# 3. Enable HTTP service:
#    - Turn HTTP ON
#    - (Optional) Edit default page:
#         <h1>Welcome to Company Local Website!</h1>
